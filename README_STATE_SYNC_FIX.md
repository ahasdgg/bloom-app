# 🔴 Исправление синхронизации состояния `currentActivity`

## 📌 Краткое описание

Исправлена **критическая архитектурная проблема** с синхронизацией состояния активности в приложении "Breath of Fresh Air".

**Проблема**: Стейл-дата при навигации, меню не реактивно, активность не завершается автоматически.

**Решение**: Добавлен `currentActivity` в store, глобальный таймер в App.tsx, реактивное меню.

**Результат**: ✅ Все 7 проблем исправлены, система работает корректно.

## ✨ Ключевые улучшения

### 1. Реактивное меню
- Иконка 🔴 пульсирует при активной активности
- Иконка ⭕ показывается, когда нет активности
- Подсказка показывает название текущей активности

### 2. Глобальный таймер
- Работает везде, не только в ActiveScreen
- Проверяет истечение времени каждую секунду
- Автоматически завершает активность

### 3. Синхронизация при навигации
- Таймер продолжает работать при переходе между экранами
- Данные не теряются
- Состояние синхронизируется при каждой навигации

### 4. Жёсткий сброс
- Завершённая активность полностью удаляется из памяти
- Состояние полностью очищается
- Нет orphaned данных
