## ВОЗМОЖНЫЕ ПРОБЛЕМЫ И РЕШЕНИЯ

### Проблема: Иконка 🔴 не пульсирует

**Решение**:
1. Проверьте, что Navigation.tsx подписана на store
2. Проверьте CSS анимацию в Navigation.css
3. Откройте DevTools → Elements → найдите `.nav-icon.pulse`

### Проблема: Таймер не синхронизируется при навигации

**Решение**:
1. Проверьте, что App.tsx имеет `useEffect` с `location` зависимостью
2. Проверьте, что `globalSyncActiveActivities()` вызывается
3. Откройте DevTools → Console → проверьте `useAppStore.getState().activeActivities`

### Проблема: Активность не завершается автоматически

**Решение**:
1. Проверьте, что App.tsx имеет глобальный таймер
2. Проверьте, что `checkExpiredActivities()` вызывается каждую секунду
3. Откройте DevTools → Console → проверьте время в `startedAt`

### Проблема: Состояние не очищается при завершении

**Решение**:
1. Проверьте, что `clearCurrentActivity()` вызывается в `handleAutoComplete()`
2. Проверьте, что `clearCurrentActivity()` вызывается в `handleContinue()` (CompletionScreen)
3. Откройте DevTools → Console → проверьте `useAppStore.getState().currentActivity`
